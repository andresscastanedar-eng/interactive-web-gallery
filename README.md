# Fermentation Project - Andrés Castañeda

## About the Project

This project is an **interactive web application** developed as part of a study on the fermentation process.

Its main goal is to **simulate data collection** and visually represent the different stages of the process through a clean and modern user interface.

The system includes:

* A registration form for user input
* A dynamic gallery documenting the experimental process with real images
