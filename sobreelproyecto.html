Sobre el proyecto

Este proyecto consiste en una aplicación web interactiva desarrollada como parte de un estudio sobre el proceso de fermentación. Su objetivo es simular la recolección de datos y mostrar visualmente las distintas etapas del proceso mediante una interfaz clara y moderna.

El sistema incluye un formulario de registro donde el usuario ingresa información básica, así como una galería dinámica que documenta el proceso experimental con imágenes reales.

Tecnologías Utilizadas

  • HTML para la estructura del sitio.
  • CSS3 para el diseño visual y experiencia del usuario (Implementando Bootstrap)
  • JavaScript para la interactividad del sitio.

Habilidades aplicadas

  • Diseño de interfaces limpias y funcionales.
  • Organización de contenido y jerarquía visual.
  • Manipulación del DOM con JavaScript.
  • Creación de formularios interactivos.
  • Desarrollo de componentes reutilizables.

Mi rol en el proyecto

Desarrollé completamente la interfaz del sitio, incluyendo el diseño visual, la estructura del formulario y la implementación de la galería interactiva.
